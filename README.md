# Casheer Auth Service

Authentication and Tenant management service for Casheer POS system.

## Features

- 🔐 User authentication (login/register)
- 🏪 Tenant/store management
- 👥 User management (admin/cashier)
- 🖼️ Logo upload to Cloudinary
- 📝 Store settings and configuration
- 🔑 JWT-based authentication

## Tech Stack

- Go 1.21
- Fiber v2 (web framework)
- GORM (ORM)
- PostgreSQL (database)
- Cloudinary (image storage)
- JWT (authentication)

## Prerequisites

- Go 1.21 or higher
- PostgreSQL 14 or higher
- Cloudinary account (for image upload)
